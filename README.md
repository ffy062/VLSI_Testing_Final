# Using Git
## Create a branch
``` 
 $ git branch <branchname>
```
- Create and got to the branch directly
```
 $ git checkout -b <branchname>
```
## Go to the branch
```
$ git checkout <branch> 
```
## List all branchs
```
 $ git branch
```
## Add modified files
- Add all files
```
 $ git add .  
```
- Add file by name
```
 $ git add \<file1> \<file2> ... 
```
## Make a commit
```
 $ git commit -m "commit message"
```
## Merge branch
```
 $ git checkout main
```
- Remember to pull form remote before merge
```
 $ git pull 
 $ git merge <branch>
```
## Push to remote
```
 $ git push
```
## Delete branch
```
 $ git branch -d \<branch>
```
