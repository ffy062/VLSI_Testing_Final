# Using Git
## Create a branch
- $ git branch \<branchname>
- $ git checkout -b \<branchname>
## Go to the branch
- $ git checkout \<branch> 
## List all branchs
- $ git branch
## Merge branch
- $ git checkout main
- $ git merge \<branch>
## Delete branch
- $ git branch -d \<branch>
